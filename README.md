# Diário 10 📰

- Uma aplicação web desenvolvida em **React** que consome a **GNews API** para exibir notícias em tempo real, permitindo filtragem por idioma, país e categoria.  
- Projeto criado para praticar consumo de APIs, componentização, estado global e estilização com Tailwind.
- **Para visualizar**: https://news-page-red.vercel.app/

---

## Tecnologias utilizadas

- **React** — criação da interface e componentes reutilizáveis  
- **Vite** — build rápido para front-end moderno  
- **TailwindCSS** — estilização ágil e responsiva  
- **GNews API** — fonte das notícias  
- **Axios** — consumo da API  
- **React Hook Form** — gerenciamento de formulários  
- **Lucide Icons** — ícones modernos  
- **React Router** — navegação entre páginas  

---

## Funcionalidades

- Buscar notícias em tempo real  
- Filtro por:
  - País
  - Idioma
  - Categoria
- Renderização dinâmica por componentes (NewsBox, Header, etc.)
- Tratamento de erros da API  
- Layout responsivo usando Tailwind  
- Loading states e UX aprimorado
